// Currency routes 
POST /api/currency - Добавление валюты
в json формате указываем поля (name, code, sign)
{
    "name": "Dollar",
    "code": "USD",
    "sign": "$"
}

GET /api/currencies - Получение списка всех валют
GET /api/currency/{code} - Получение валюты по её коду 



// Exchangerate cuurency routes
POST /api/exchangeRates - Добавдление обменного курса 
в json формате указываем поля (baseCurrencyCode, targetCurrencyCode, rate)
{
    "baseCurrency": "USD",
    "targetCurrency": "ERU",
    "rate": 0.99
}

GET /api/exchangeRates - Получение всех обменных курсов
GET /api/exchangeRates/{codes} - Получение определнного обменного курса
PATCH /api/exchangeRates/{codes} - Обновление рейта в обменном курсе 
GET /api/exchangeRates/exchange?from=BASE_CURRENCY_CODE&to=TARGET_CURRENCY_CODE&amount=$AMOUNT - Получение суммы вычисленной с помощью обменного курс
